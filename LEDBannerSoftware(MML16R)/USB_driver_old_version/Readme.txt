                     Installing Introduction

you may use one of the following methods to install usb driver.

1. To install usb driver in the windows, you may simple run the PreInstaller.exe in the directory of Win_PreInstall. 
    Then plug usb connector into you PC usb port. 

2. Plug the usb connector into PC usb Port, the windows will detect new device and pop up dialogue, then specify 
    the location and proper driver file in the directory of Win.
